## 🧪 このリポジトリについて

このリポジトリは、ブドウちゃんが開発・実験している各種Botのコードを管理する実験場です。  
Discord / LINE / Web3 / AI を活用した自動化ツールや遊びの仕組みを試しています。

## 📋 開発環境

- Python 3.11+
- Discord.py
- FastAPI
- SQLAlchemy
- Poetry (パッケージ管理)
- Git

## 🚀 セットアップ方法

1. Poetryのインストール
2. `poetry install` で依存関係のインストール
3. `.env` ファイルの設定
4. `poetry run python main.py` で実行
