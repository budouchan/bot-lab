# GlucoseMan Reaction Points Bot

このボットはDiscordのリアクション（グルコースマンのマーク）に対してポイントを付与する機能を持つDiscordボットです。

## リアクションポイントシステム

### リアクションとポイント
- 🧃（グルコースマン）: 情報提供のお礼 - 1pt
- 🍏（さおりんご）: 情報が採用された場合の印 - 1pt
- 🍇（ブドウちゃん）: 写真などの素材を提供した場合 - 1pt

### 注意点
- 同じ投稿に対して同じタイプのリアクションは1回限り
- 🍏（さおりんご）と🍇（ブドウちゃん）は、ブドウちゃんかさおりんごのみが付けることができます
- リアクションを外すとポイントも減少します

## コマンド

### ランキング表示
```
/rank
```
上位10人のポイントランキングを表示します。

### 個人ポイント確認
```
/points @ユーザー名
```
指定されたユーザーのポイント詳細を表示します。

## セットアップ

1. 必要なパッケージのインストール:
```bash
pip install discord.py python-dotenv
```

2. `.env`ファイルにDiscord Bot Tokenを設定:
```
DISCORD_BOT_TOKEN=あなたのトークン
```

3. ボットを起動:
```bash
python3 glucose_bot.py
```

## ファイル構成

- `glucose_bot.py`: メインのボットコード
- `points.json`: ユーザーポイントデータの保存
- `.env`: 環境変数（Discord Bot Token）
- `README.md`: ドキュメント
